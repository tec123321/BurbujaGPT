# Avisos de terceros

## SuperWindow

Globo WhatsApp V2 contiene una adaptación independiente basada en ideas arquitectónicas y porciones del proyecto de código abierto **SuperWindow** de eiyooooo.

- Fuente: https://github.com/eiyooooo/SuperWindow
- Licencia: GNU General Public License v3.0
- Revisión examinada: `9358ce75ae7e20eeea99e0e9a0ea92496eaa919a`

La adaptación se concentra principalmente en:

- `SystemDisplayEngine.java`
- `VirtualDisplaySessions.java`
- `ShizukuDisplayBridge.java`
- `ShellDisplayCreator.java`

Los cambios específicos incluyen integración con burbujas Android, ejecución de WhatsApp, apertura de conversaciones mediante el `PendingIntent` original, persistencia de una pantalla compartida, diagnósticos para One UI y rutas alternativas para mover tareas e inyectar entrada.
